Lab 6: LaTeX

In this lab, we learned the basic syntax of LaTeX and created our LaTeX document.

Main content

This part mainly records the exercises in lab:
1. We learned the basic layout of a TeX file and knew comments are denoted by a preceding % symbol. And we studied the documentclass command, which has many options include article, report, book, letter.
2. We tried to create sections within our documents using \section and \subsection commands. And we also tried to add an appendices section using appendix command. Then we created a numbered list and a bullet pointed list using enumerate and itemize respectively.
3. We needed to import a package called graphic using the \usepackage graphicx command. Then we could insert a graphic using \includegraphics[width=XXmm]{Path/To/File.png} command. Besides, we learned that using the center environment could also achieve similar result.
4. We tried to insert mathematical formula in our document. And we learned how to write an algorithm using LaTeX. To start writing an algorithm we needed to create an algorithmic environment using \begin{algorithmic} ... \end{algorithmic} and import a package called algorithmic using usepackage algorithmic. Then we learned how to reference the equations and figures we have produced.
